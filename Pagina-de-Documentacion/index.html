Introduccion

Python es un lenguaje de programación de alto nivel cuya máxima es la legibilidad del código. Las principales características de Python son las siguientes:

  • Es multiparadigma, ya que soporta la programación imperativa, programación orientada a objetos y funcional.
  • Es multiplataforma: Se puede encontrar un intérprete de Python para los principales sistemas operativos: Windows, Linux y Mac OS. Además, se puede reutilizar el mismo código en cada una de las plataformas.
  • Es dinámicamente tipado: Es decir, el tipo de las variables se decide en tiempo de ejecución.
  • Es fuertemente tipado: No se puede usar una variable en un contexto fuera de su tipo. Si se quisiera, habría que hacer una conversión de tipos.
  • Es interpretado: El código no se compila a lenguaje máquina.
Tipos de Datos

Los tipos de datos son los siguientes:

  • Int (Entero): Es un tipo de dato numérico sin decimales, utilizado para representar números enteros. En la mayoría de los lenguajes de programación, los enteros son almacenados con un tamaño fijo en bits (por ejemplo, 32 bits o 64 bits) y pueden ser positivos, negativos o cero. En Python, los enteros se pueden declarar sin especificar un tipo y se pueden usar en operaciones matemáticas como la suma, la resta, la multiplicación y la división.
    • x = 42 print(x) # Imprime 42
  • Boolean: Es un tipo de dato lógico que solo puede tomar dos valores: verdadero o falso. Es útil para representar valores de verdad o de decisión en la programación. Por ejemplo, una variable booleana puede ser usada para indicar si un usuario ha iniciado sesión o no. En Python, las variables booleanas se pueden declarar usando palabras clave como True y False.
    • is_logged_in = True print(is_logged_in) # Imprime True
  • String (Cadena de caracteres): Es un tipo de dato que representa una secuencia de caracteres (letras, números, símbolos, etc.). Las cadenas de caracteres se pueden usar para representar texto, por ejemplo, para imprimir mensajes en la pantalla o para almacenar información de texto en una variable. En Python, las cadenas de caracteres se pueden declarar entre comillas simples o dobles.
    • name = "John Doe" print(name) # Imprime John Doe
  • Float (Número flotante): Es un tipo de dato numérico con decimales, utilizado para representar números reales con una precisión finita. Los números flotantes se almacenan en la computadora en formato binario, por lo que pueden haber diferencias entre la representación de un número en la computadora y su representación decimal. En Python, los números flotantes se pueden declarar con un punto decimal.
    • pi = 3.14 print(pi) # Imprime 3.14
  • Listas: Son un tipo de dato que permite almacenar varios valores de diferentes tipos en una sola estructura de datos. Las listas se pueden usar para almacenar una colección de valores relacionados, como por ejemplo, los nombres de los días de la semana o los precios de una lista de productos. En Python, las listas se pueden declarar usando corchetes y separando los elementos de la lista con comas.
    • colors = ["red", "green", "blue"] print(colors) # Imprime ['red', 'green', 'blue']
Variables

En Python, las variables se utilizan para almacenar información que se puede utilizar posteriormente en el programa. Se pueden almacenar diferentes tipos de datos, como números, cadenas de texto, listas, diccionarios, etc.

Para declarar una variable en Python simplemente se asigna un valor a un nombre:

nombre_de_variable = valor
Operadores

Los operadores en Python son los signos matemáticos que permiten realizar operaciones aritméticas y comparar valores. Algunos de los operadores más comunes en Python son:

+ (suma), - (resta), * (multiplicación), / (división), % (módulo), = (asignación), == (igualdad), != (desigualdad), > (mayor que), >= (mayor o igual que), < (menor que), <= (menor o igual que)
Condicionales

Los condicionales en Python permiten ejecutar ciertas acciones solo si se cumple una determinada condición. La estructura básica de un condicional es:

  • if
  •       (acciones a realizar si se cumple la condición)
  • else (acciones a realizar si no se cumple la condición):
if x > 0: print("x es un número positivo") else: print("x es un número negativo o cero")
Bucles

Los bucles son una herramienta esencial en programación, que permiten ejecutar un bloque de código varias veces. Python tiene dos tipos principales de bucles: el bucle "for" y el bucle "while".

El bucle "for" se utiliza para iterar a través de una secuencia, como una lista o una cadena de texto. Por ejemplo:

  • for elemento in secuencia:
  • # código a ejecutar para cada elemento en la secuencia
for i in range(5): print(i) # Imprime: # 0 # 1 # 2 # 3 # 4

El bucle "while" se utiliza para repetir un bloque de código mientras se cumpla una condición. Por ejemplo:

  • while condición:
  • # código a ejecutar mientras se cumpla la condición
i = 0 while i < 5: print(i) i += 1 # Imprime: # 0 # 1 # 2 # 3 # 4
Funciones

Las funciones son una parte esencial de la programación en Python, ya que permiten organizar y reutilizar el código. Las funciones en Python se definen con la palabra clave "def".

Por ejemplo, aquí hay una función que toma dos números y devuelve su suma:

def suma(a, b): return a + b resultado = suma(3, 4) print(resultado) # Imprime: 7
